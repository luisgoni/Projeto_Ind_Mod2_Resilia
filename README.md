# Aplicativo de Verificação de Compatibilidade de Candidatos

# Este é um aplicativo desenvolvido para verificar a compatibilidade de candidatos com uma vaga de acordo com seus resultados em diferentes etapas de um processo seletivo.

# ## Funcionalidades

# - Armazenamento de resultados de candidatos em formato específico (por exemplo, "eX_tX_pX_sX").
# - Busca de candidatos com notas maiores ou iguais às mínimas especificadas pelo usuário.

# ## Como Usar

# 1. Defina os resultados dos candidatos na lista `aprovados` no formato desejado.
# 2. Execute a função `buscar_candidatos` informando as notas mínimas desejadas para cada etapa.
# 3. Os candidatos compatíveis serão retornados e podem ser utilizados para outras finalidades, como exibição na interface do aplicativo.
